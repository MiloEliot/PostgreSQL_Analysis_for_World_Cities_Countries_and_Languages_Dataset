# PostgreSQL Analysis - World cities, countries and languages

This project explores the world cities, countries and languages dataset from Kaggle using Postgresql. 

It demonstrates: 
- Basic SQL commands 
- Window functions
- Customized SQL functions
- Anonymous Do blocks

## Dataset
The original dataset can be found at: [Kaggle](https://www.kaggle.com/datasets/adilshamim8/world-cities-countries-and-languages-dataset/data). 


## Repository Structure
- `data` - the csv files downloaded from Kaggle
- `sql` - SQL script used for database construction and exploratory data analysis
- `outputs` - the generated plots
